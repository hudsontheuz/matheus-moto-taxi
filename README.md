# Matheus Moto Táxi

Landing page mobile-first para serviço de moto táxi e entregas em Pato Branco/PR e região.

## Stack

- Astro
- Tailwind CSS
- TypeScript
- HTML semântico
- SEO local
- Mobile-first

## Como rodar

```bash
npm install
npm run dev
```

Depois acesse o endereço mostrado no terminal, normalmente:

```bash
http://localhost:4321
```

## Build de produção

```bash
npm run build
npm run preview
```
